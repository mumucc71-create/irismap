상담요청 연동 설정

구글 Apps Script 배포 URL을 저장하면 쇼핑 상담요청이 구글시트에 저장되고 mumu5499@hanmail.net으로 자동 이메일이 발송됩니다.

쇼핑 화면

설정 순서

1. 구글시트 하나를 만듭니다.

2. 확장 프로그램 → Apps Script를 엽니다.

3. consult-google-apps-script.gs 내용을 붙여넣고 저장합니다.

4. 배포 → 새 배포 → 웹 앱으로 배포합니다. 실행 권한은 본인, 액세스 권한은 모든 사용자로 설정합니다.

5. 발급된 웹앱 URL을 이 화면에 붙여넣고 저장합니다.
